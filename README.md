# MoonGame

Made for the Game Makers Iceland Summer Game Jam 2026; theme was Eclipse https://valgardg.itch.io/moon-simulator ☀️

Solo dev 🚀

Built with Unity 💻

Game description:
- You are the moon, and you orbit the earth. For some odd reason, the sun also orbits the earth, and is tiny, but you don't think about that because you're the moon. 

- As a particularly sentient moon, you know you'll continue to orbit the earth as you have for a long time now. But you also get a little bored from time to time, because orbiting the earth for eternity does that. Luckily for you, you've figured out how to pass the time. 

- You've figure out that aligining yourself between the sun and earth is fun! Why is this so fun you may ask? Well, because the earth starts to absolutely freak out for whatever reason!
